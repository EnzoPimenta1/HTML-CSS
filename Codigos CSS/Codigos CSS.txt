DISPLAY = A propriedade display no CSS define como um elemento HTML é renderizado e se comporta no layout
block: Elementos que ocupam toda a largura disponível e iniciam em uma nova linha (ex: <div>, <p>, <h1>).
inline: Elementos que ficam na mesma linha, ocupando apenas o espaço necessário, sem quebra de linha (ex: <span>, <a>, <strong>).
inline-block: Combina o fluxo inline (na mesma linha) com as propriedades de block (aceita width, height, margin vertical).
none: Remove completamente o elemento do fluxo da página, como se não existisse (diferente de visibility: hidden, que mantém o espaço).
flex (Flexbox): Transforma o elemento em um contêiner flexível, ideal para alinhar e distribuir itens em uma dimensão (linha ou coluna).
grid (CSS Grid): Transforma o elemento em um contêiner de grade, permitindo layouts bidimensionais complexos (linhas e colunas).
table: Faz o elemento se comportar como uma tabela HTML (funcionalidade mais antiga).
list-item: Faz o elemento se comportar como um item de lista (<li>).

PSEUDO-CLASSES DE ESTADO DO USUARIO (Interativas)
Estas ativam estilos baseados na interação do usuário com o elemento. 
:hover: Aplicado quando o cursor do mouse passa sobre um elemento.
:active: Aplicado no momento em que um elemento é ativado (ex: botão pressionado/clicado).
:focus: Aplicado quando um elemento está focado (geralmente via teclado com Tab ou clique).
:visited: Estiliza links que o usuário já visitou.
:link: Estiliza links que ainda não foram visitados. 
:root = Sua principal função é definir variáveis CSS globais (propriedades personalizadas)

Pseudo-elementos em CSS são palavras-chave adicionadas a um seletor que permitem estilizar partes específicas de um elemento ou inserir conteúdo "falso" (que não existe no HTML) antes ou depois do conteúdo original. 
::before e ::after (Os mais utilizados) 
Inserem conteúdo (texto, imagem, ícones) antes (::before) ou depois (::after) do conteúdo de um elemento.

> = O símbolo > em CSS é conhecido como Seletor de Filho Direto (ou Child Combinator). Ele é utilizado para aplicar estilos a um elemento que é filho direto de outro, sem afetar netos ou descendentes mais distantes na estrutura HTML.
