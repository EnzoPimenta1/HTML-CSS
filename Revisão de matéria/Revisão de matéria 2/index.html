Revisão de matéria

Negrito / Destaque

Para adicionar um termo em destaque usamos < strong >

Itálico / Ênfase

Para adicionar um termo em itálico usamos < em >

Texto marcado

Para adicionar um texto marcado usamos < mark >

Para marcar todos os textos com a mesma cor usamos < style >, e dentro de style usamos: mark {background-color:}

Texto deletado

Podemos também marcar um texto excluido para mostrar que ele deve ser lido, porém, não considerado < del >

Texto inserido

Podemos também marcar um texto inserido para mostrar que ele foi colocado depois! < ins >

Texto sobrescrito

Para inserir coisas do tipo: x 20 + 3 < sup >

Texto subscrito

Para inserir coisas do tipo: H20 < sub >
